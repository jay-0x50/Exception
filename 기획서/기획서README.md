# Exception 기획 문서

`Exception`은 프로그래밍 언어와 개발 도구의 아이콘/마스코트를 보스 캐릭터로 재해석한 1:1 보스 레이드 액션 데모입니다.

## 문서 구조

| --- | --- |
| `00_프로젝트_개요` | 프로젝트 핵심 기획서, 게임 루프, MVP 범위 |
| `01_전투_시스템` | 판정, 리소스, 그로기, 처형 규격 |
| `02_플레이어` | 락온 카메라, 피격 리액션, 플레이어 UX |
| `03_보스` | 보스 콘셉트, 패턴, Intro 연출 |
| `04_월드_흐름` | 체크포인트, 리스폰, 재진입 동선 |
| `05_기술_구현` | Unreal C++ 구조, AI 설계, 구현 체크리스트 |

## MVP 기준

- 필드 1개, 체크포인트 1개, 보스방 1개, 보스 1체
- 플레이어 기본 액션: 약공격, 강공격, 회피, 패링, 처형
- 보스 기본 구조: Phase 1/2, 그로기, 처형, 리스폰 초기화
- 포트폴리오 핵심: C++ 컴포넌트 구조, 인터페이스 기반 전투, 이벤트 기반 UI, Behavior Tree AI
