## 1. 공통 규칙

- 모든 패턴은 아래 요소를 반드시 포함:
    - **텔(예고 동작)**: 플레이어가 “피해야 한다”는 것을 인지 가능
    - **위험 구간(히트 프레임)**: 판정이 실제로 발생하는 구간
    - **리커버리(후딜)**: 대응 성공 시 보상(딜 찬스)을 만드는 구간
- 공격 타입 태그 예시:
    - `Normal` (패링 가능)
    - `Smash` (패링 불가, 회피 권장)
    - `AOE` (범위기, 위치 이동 유도)

## 1-1. 패턴 발동 철학 (AI 기반)

본 보스는 “쿨타임=바로 사용” 혹은 “고정 로테이션”이 아니라,

**쿨타임은 밸런스 장치로만 사용하고, 실제 발동 타이밍은 AI가 판단**하는 구조를 목표로 함.

### 운영 규칙

- 패턴은 쿨타임이 끝나면 **Ready 상태**가 됨
- Ready 상태여도 즉시 실행하지 않고, 아래 조건을 만족할 때 실행:
    - 사거리/각도 유리
    - 플레이어가 공격 지속/스태미나 고갈 등으로 회피 여지가 낮음(추정)
    - 직전 패턴과 중복되지 않음(반복 페널티)

### 선택 방식(권장)

- 후보 패턴들 중 조건을 만족하는 것만 모아 **가중 랜덤(Weighted Random)**으로 선택
- `LastUsedPattern` / 최근 N초 사용 패턴 기반으로 가중치 감소
- 필요 시 “확정 히트 기회”가 큰 패턴(예: Gap closer)을 상위 우선순위로 배치

## 2. Phase 1 패턴

### P1-1) 기본 2연격 (Normal)

- **Trigger:** 거리 < 250
- **Cooldown:** 3s
- **텔:** 무기 들어올리며 0.3s 멈칫
- **히트:** 1타(0.15s) → 2타(0.15s)
- **특이사항:** 1타 후 거리 재검사 → 사거리 유지 시 2타 연계
- **대응 의도:** 1타 회피 후 2타 패링(고보상) 또는 연속 회피(안전)

### P1-2) 대지 분쇄 (AOE / Smash)

- **Trigger:** 250~600
- **Cooldown:** 8s
- **텔:** 무기 높게 들고 바닥 조준(0.6s)
- **히트:** Notify 프레임에 충격파 AOE 스폰 + Radial Damage
- **리커버리:** 긴 후딜(예: 1.0s)로 반격 창 제공
- **대응 의도:** “한 번 크게 피하고 딜한다”를 학습

## 3. Phase 2 패턴

### P2-0) 광폭화 진입 (연출)

- **Trigger:** HP <= 50% (최초 1회)
- **효과:** 잠깐 무적 + 포효 + 이펙트/사운드 강화
- **BT:** 현재 태스크 Abort(Self) 후 P2 서브트리 활성화

### P2-1) 그림자 돌진 (Gap closer)

- **Trigger:** 거리 > 500
- **Cooldown:** 6s
- **텔:** 자세 낮추고 잔상 예고(0.4s)
- **히트:** 돌진 경로에 히트 박스(또는 캡슐 스윕)
- **특이사항:** 시작 시 플레이어의 최종 위치/방향 저장 후 추적 보간
- **대응 의도:** “측면 회피”를 요구, 성공 시 보스가 잠깐 오버슈트하여 딜 찬스 제공

## 4. 패턴 밸런싱 파라미터(권장)

- 텔 길이 (초)
- 공격 유효 프레임(초)
- 후딜(초)
- 이동 거리 / 추적 강도
- 카메라/사운드 강조 강도(학습 지원)
- Damage / GroggyDamage / HitStop

- 체력 20% 이하 “마무리 패턴” 1개 (클라이맥스)
