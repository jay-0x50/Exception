## 1. 비주얼 아이덴티티

## 2. 캐릭터 비주얼 가이드

### 2-1) Handel (플레이어)

| 항목 | 내용 |
|---|---|
| **실루엣** | 날렵한 인간형. 후드 코트 + 경량 아머 조합 |
| **컬러** | 다크 그레이 베이스 + 시안 블루 포인트 (관절, 눈) |
| **특징** | 몸 어딘가에 코드 라인이 흐르는 듯한 발광 문양 |
| **무기** | 단검 또는 짧은 한손검 (빠른 전투 스타일 표현) |
| **에셋 방향** | Mixamo 기본 인간형 + 직접 제작 아머 텍스처 오버레이 |

**상태별 비주얼 변화**

| 상태 | 변화 |
|---|---|
| 기본 | 시안 발광 문양 안정적으로 흐름 |
| HP 30% 이하 | 발광 문양 깜빡임, 노이즈 이펙트 |
| 그로기 처형 중 | 발광 문양 최대 밝기, 슬로우모션 연출 |

### 2-2) Nel

| 항목 | 내용 |
|---|---|
| **실루엣** | 작고 가벼운 인간형. 망토 또는 긴 로브 |
| **컬러** | 화이트 + 소프트 블루 그라데이션. 반투명한 느낌 |
| **특징** | 몸이 완전히 실체화되지 않은 느낌 (Null = 정의되지 않음). 가장자리가 흐릿하게 퍼짐 |
| **눈** | 빛이 없는 흰 눈. 감정이 생기면 서서히 색이 생김 |
| **에셋 방향** | 무료 에셋 기본형 + 반투명 머티리얼 직접 제작 |

**감정 상태별 비주얼**

| 상태 | 눈 색상 | 몸 투명도 |
|---|---|---|
| 기본 | 흰색 (무표정) | 70% 불투명 |
| 감정 생길 때 | 소프트 블루 | 85% 불투명 |
| 사라지기 직전 | 서서히 투명 | 0%로 페이드 |

### 2-3) Terminal

| 항목 | 내용 |
|---|---|
| **실루엣** | Handel과 동일한 구조. 하지만 손상되고 왜곡된 형태 |
| **컬러** | 다크 레드 + 블랙. 균열에서 크림슨 빛이 새어나옴 |
| **특징** | Handel의 시안 발광 문양이 레드로 변색. 일부 문양이 끊겨있음 |
| **분위기** | 처음 등장 시 실루엣만. 최종전에서 전체 모습 드러남 |
| **에셋 방향** | Handel 에셋 재활용 + 컬러/셰이더 변경으로 차별화 |

## 3. 보스 비주얼 가이드

### 3-1) Vethara (파란 드래곤)

| 항목 | 내용 |
|---|---|
| **실루엣** | 거대한 뱀형 드래곤. 날개 없음. 길고 유연한 몸통 |
| **스케일** | 플레이어 키의 약 5~6배 |
| **컬러** | 코발트 블루 비늘 + 전신에 흐르는 번개 룬 문양 |
| **눈** | 얼어붙은 듯한 흰 번개빛 |
| **특징** | 움직일 때마다 전기 파티클 발생. 얼음 결정이 비늘 사이에 박혀있음 |
| **에셋 방향** | Fab 무료 드래곤 에셋 + 직접 제작 머티리얼/파티클 |

### 3-2) Aurathos (골드 드래곤)

| 항목 | 내용 |
|---|---|
| **실루엣** | Vethara와 동일 구조. 더 두껍고 묵직한 느낌 |
| **스케일** | 플레이어 키의 약 6~7배 (Vethara보다 약간 큼) |
| **컬러** | 균열 황금 비늘 + 관절/흉부에서 용암 균열 발광 |
| **눈** | 황금빛 화염 |
| **특징** | 걸을 때 바닥에 용암 균열 생성. 숨쉴 때 불꽃 파티클 |
| **에셋 방향** | Vethara 에셋 재활용 + 컬러/머티리얼 변경 |

**Vethara + Aurathos 듀얼 배치**

```
[보스방 입장 시]

       Vethara (파랑)        Aurathos (골드)
            ●                    ●
            
                  [플레이어]
                      ▲
```
- 두 보스가 대칭으로 배치된 원형 공간
- 중앙 바닥에 Python 들여쓰기 라인 패턴

### 3-3) Fatal Elephant

| 항목 | 내용 |
|---|---|
| **실루엣** | 거대한 코끼리형. 갑옷 조각들이 몸에 박혀있음 |
| **스케일** | 플레이어 키의 약 8배. 묵직한 압박감 |
| **컬러** | 다크 그레이 + 퍼플 균열 발광 |
| **특징** | 몸 전체가 오래된 코드처럼 노이즈/글리치 이펙트. 움직일 때 바닥 균열 |
| **에셋 방향** | Fab 무료 대형 몬스터 에셋 + 직접 제작 글리치 셰이더 |

## 4. UI/UX 설계

### 4-1) HUD 전체 레이아웃

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  [플레이어 HP]          [보스 HP — Vethara]         │
│  [스태미나    ]          [보스 HP — Aurathos]        │
│  [그로기 게이지]                                    │
│                                                     │
│                                                     │
│                   [ 게임 화면 ]                     │
│                                                     │
│                                                     │
│                                        [Nel 아이콘] │
│                                        [Nel 대사창] │
└─────────────────────────────────────────────────────┘
```

### 4-2) 플레이어 HUD

| 항목 | 위치 | 디자인 방향 |
|---|---|---|
| **HP 바** | 좌측 상단 | 시안 블루. 감소 시 붉게 플래시 |
| **스태미나 바** | HP 바 아래 | 흰색/회색. 소모 시 빠르게 감소, 회복 시 부드럽게 충전 |
| **그로기 게이지** | 스태미나 아래 | 네온 옐로우. 누적될수록 밝아짐. 발동 시 전체 플래시 |
| **상태 아이콘** | HP 바 옆 | 회피 무적 중 / 패링 판정 중 아이콘 표시 |

**HP 바 상태별 컬러**

| HP 비율 | 색상 |
|---|---|
| 100% ~ 50% | 시안 블루 `#00D4FF` |
| 50% ~ 30% | 오렌지 `#FF8C00` |
| 30% 이하 | 레드 + 깜빡임 `#FF2D55` |

### 4-3) 보스 HP UI

| 항목 | 내용 |
|---|---|
| **위치** | 화면 상단 중앙 |
| **구성** | 보스 이름 + HP 바 (Vethara / Aurathos 개별 표시) |
| **연동 경고** | 두 HP 차이 30% 이상 시 체력 많은 쪽 바 테두리 붉게 깜빡임 (격노 경고) |
| **등장 애니** | 보스 Intro 중반 페이드 인 (0.5s) |
| **폰트** | Bebas Neue 대문자 |

### 4-4) Nel 대사창

| 항목 | 내용 |
|---|---|
| **위치** | 화면 우측 하단 |
| **구성** | Nel 아이콘 (반투명) + 모노스페이스 타이핑 텍스트 |
| **등장 방식** | 타이핑 이펙트 (터미널 입력 느낌) |
| **히든 루트 부탁 표시** | Nel 아이콘 옆 `[?]` 마커. 들어주면 `[✓]`로 변경 |

### 4-5) 처형 가능 프롬프트

| 상태 | 표시 |
|---|---|
| 보스 그로기 + 범위 밖 | 보스 머리 위 흐린 아이콘 |
| 보스 그로기 + 범위 안 | 보스 머리 위 시안 처형 아이콘 + `[E] 처형` 텍스트 |
| 처형 중 | 전체 UI 숨김 |

### 4-6) 시스템 로그 UI (Terminal 등장 시)

Terminal이 로그를 보낼 때 화면 우측 상단에 터미널 창 형식으로 표시.

```
┌─────────────────────────┐
│ [SYSTEM LOG]            │
│ > 처리 중. 예상보다     │
│   빠르군.               │
│ > 계속해라, Handel.     │
└─────────────────────────┘
```

- 폰트: JetBrains Mono
- 배경: 반투명 블랙
- 텍스트 컬러: 크림슨 레드 `#FF2D55`
- 타이핑 이펙트 적용

### 4-7) 메인 메뉴

| 항목 | 내용 |
|---|---|
| **배경** | 타이틀 배경 이미지 (기획서 이미지 활용) + 글리치 파티클 |
| **로고** | `EXCEPTION` 대문자 + 하단에 작게 `an unhandled exception has occurred` |
| **메뉴 항목** | `NEW GAME` / `CONTINUE` / `SETTINGS` / `QUIT` |
| **폰트** | 로고: Rajdhani / 메뉴: JetBrains Mono |
| **커서** | `>` 커서가 항목 앞에서 깜빡임 (터미널 스타일) |

### 4-8) 사망 화면

```
[화면 페이드 블랙]

        YOU DIED
    
    [재시도]  [메인 메뉴]
```

- 폰트: Bebas Neue
- 컬러: 다크 레드 → 흰색 페이드 인
- 하단에 작게 Terminal의 시스템 로그 텍스트 랜덤 표시

**사망 시 랜덤 로그 텍스트 (예시)**
```
[PROCESS TERMINATED]
[EXCEPTION NOT HANDLED]
[RETRY? Y/N]
[INSTANCE RELOAD IN PROGRESS...]
```

## 5. 아이템 & 소모품

> MVP 기준 아이템은 **최소화**. 전투 리듬을 방해하지 않는 선에서만.

### 5-1) 소모품 목록

| 아이템 | 이름 | 효과 | 획득 방법 | 최대 소지 |
|---|---|---|---|---|
| **힐 패킷** | `HEAL.exe` | HP 40 회복 | 체크포인트에서 보충 | 3개 |
| **스태미나 패킷** | `STAMINA.exe` | 스태미나 즉시 전량 회복 | 체크포인트에서 보충 | 2개 |
| **그로기 부스터** | `OVERFLOW.exe` | 다음 공격 그로기 데미지 2배 (1회) | 보스 처치 보상 | 1개 |

**디자인 방향**
- 포션 형태가 아닌 **USB 드라이브 / 데이터 카드** 형태
- 사용 시 데이터 인젝션 느낌의 이펙트
- 아이템 이름이 전부 `.exe` 확장자로 끝남

### 5-2) 아이템 UI

| 항목 | 내용 |
|---|---|
| **위치** | 화면 우측 하단 (Nel 대사창 위) |
| **표시** | 아이템 아이콘 + 잔여 개수 |
| **사용 키** | `Q` (빠른 사용) |
| **쿨타임** | `HEAL.exe` 사용 후 15s 쿨타임 |

### 5-3) 보상 아이템 (보스 처치 후)

| 보스 | 보상 | 효과 |
|---|---|---|
| **Serpent.py 처치** | `SERPENT_CORE.dat` | MaxHP +10, 스태미나 회복속도 +10% |
| **Fatal Elephant 처치** | `ELEPHANT_CORE.dat` | 강공격 그로기 데미지 +5, `OVERFLOW.exe` 최대 소지 +1 |
| **Terminal 처치** | `TERMINAL_CORE.dat` | (엔딩 연출용, 실질 효과 없음) |

## 6. 월드 & 환경 디자인

### 6-1) 전체 공간 컨셉

The Runtime 내부는 **추상화된 디지털 공간**이다.
현실의 물리 법칙을 따르되, 배경에 코드/데이터 미학이 녹아있다.

| 구역 | 분위기 | 키 비주얼 |
|---|---|---|
| **체크포인트** | 안정적, 따뜻한 느낌 | 시안 발광 기둥. 바닥에 안전한 코드 라인 |
| **필드 구간** | 점점 불안정해지는 느낌 | 배경 노이즈 증가, 바닥 균열 |
| **보스방 — Python** | 원형 공간, 들여쓰기 패턴 | 바닥에 Python 코드 라인. 파랑+골드 대비 |
| **보스방 — PHP** | 압박감 있는 좁은 공간 | 바닥에 PHP 태그 패턴 `<?` `?>`. 퍼플 안개 |
| **Terminal 최종 구역** | 완전히 붕괴된 공간 | 모든 요소가 글리치. 배경이 레드+블랙 |

### 6-2) 체크포인트 디자인

| 항목 | 내용 |
|---|---|
| **형태** | 수직으로 솟은 발광 기둥 (소울라이크의 모닥불 포지션) |
