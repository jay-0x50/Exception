## 3. 공통 규칙

모든 패턴은 아래 3단계 구조를 반드시 포함한다.

- **텔(예고 동작):** 플레이어가 "피해야 한다"는 것을 인지할 수 있는 예고 모션
- **위험 구간(히트 프레임):** 실제 판정이 발생하는 구간
- **리커버리(후딜):** 대응 성공 시 딜 찬스를 만드는 구간

공격 타입 태그:

- `Normal` — 패링 가능
- `Smash` — 패링 불가, 회피 권장
- `AOE` — 범위기, 위치 이동 유도
- `Grab` — 잡기, 패링/회피 모두 가능하나 타이밍이 까다로움 (선택 패턴)

## 4. 패턴 발동 철학

쿨타임은 밸런스 안전장치일 뿐, **실제 발동 타이밍은 AI가 상황을 보고 판단**한다.

- 쿨타임 종료 = **Ready 상태** (발동 후보에 포함)
- Ready 상태에서도 아래 조건을 모두 만족해야 발동:
    - 사거리/각도가 유리한 상황
    - 플레이어가 회피 여지가 낮다고 추정될 때 (연속 공격 중, 스태미나 고갈 등)
    - 직전에 같은 패턴을 쓰지 않았을 때 (반복 페널티)
- 후보 패턴들 중 **가중 랜덤(Weighted Random)** 으로 최종 선택

## 5. Phase 1 패턴

### P1-1) Tail Sweep Combo `Normal`

> "꼬리로 휩쓸고, 이어서 몸통으로 회전 타격"

| 항목 | 수치/내용 |
| --- | --- |
| Trigger 거리 | < 250 |
| Cooldown | 3s |
| AttackType | Normal (패링 가능) |
| GroggyDamage | 1타 +8 / 2타 +12 |
| Damage | 1타 20 / 2타 20 |
| HitStop | 0.08s |
| ReactionType | Light_Hit |

**텔 (0.3s)**
- 상체를 낮추고 꼬리를 뒤로 크게 감으며 멈칫
- 꼬리가 발광 (시각 신호)

**히트 프레임**
- 1타: 꼬리 수평 스윕 `0.15s` (좌→우)
- 거리 재검사 → 사거리 유지 시 2타 자동 연계
- 2타: 몸통 회전 타격 `0.15s`

**리커버리 (0.5s)**
- 2타 후 보스가 자세를 정비하는 모션

**대응 의도**
- 1타 회피 → 2타 패링 (고보상, 그로기 +12)
- 1타·2타 연속 회피 (안전하지만 스태미나 소모)
- 리커버리 구간에 반격 가능

### P1-2) Coil Slam `AOE / Smash`

> "몸을 둥글게 압축한 뒤 낙하해 광역 충격파를 발생시킨다"

| 항목 | 수치/내용 |
| --- | --- |
| Trigger 거리 | 250 ~ 600 |
| Cooldown | 8s |
| AttackType | Smash + AOE (패링 불가) |
| GroggyDamage | 0 |
| Damage | 직격 45 / AOE 충격파 25 |
| HitStop | 0.20s |
| ReactionType | 직격 Knockback / 충격파 Heavy_Hit |
| AOE 반경 | 400 |

**텔 (0.6s)**
- 몸을 공처럼 둥글게 말아 압축하는 모션
- 바닥에 충격파 예고 원형 데칼 표시 (서서히 붉게 채워짐)

**히트 프레임**
- 낙하 착지 순간 `0.1s` 직격 판정 (착지 지점 반경 100)
- 착지 직후 `0.2s` 충격파 AOE 확장 (반경 400)

**리커버리 (1.0s)**
- 착지 후 자세 재정비. 긴 후딜로 반격 창 제공

**대응 의도**
- 예고 원 밖으로 이동 후 후딜에 반격
- "한 번 크게 피하고 딜한다"는 학습 유도

## 6. Phase 2 진입 — Enrage Sequence

> HP 50% 이하 최초 도달 시 1회 발동

| 항목 | 내용 |
| --- | --- |
| Trigger | `CurrentHP <= MaxHP * 0.5` 최초 1회 |
| 무적 구간 | 진입 연출 전 구간 (약 4s) |
| BT 처리 | 현재 태스크 `Abort(Self)` → Enrage 서브트리 실행 |
| 이후 | P2 전용 서브트리로 전환. MoveSpeed 400 → 550 |

**연출 순서**
```
보스 HP 50% 도달
    ↓
BT Abort → 무적 활성화
    ↓
포효 몽타주 재생 (AM_Boss_Enrage)
    - 두 머리가 동시에 하늘을 향해 울부짖음
    - 독색 이펙트가 전신에서 폭발
    ↓
Niagara VFX: 독 파티클 + 발광 강화
보스 스케일 소폭 증가 (시각적 광폭화 표현)
    ↓
무적 해제 → P2 BT 활성화
```

## 7. Phase 2 패턴

### P2-1) Shadow Dash `Smash`

> "잔상을 남기며 플레이어 방향으로 고속 돌진"

| 항목 | 수치/내용 |
| --- | --- |
| Trigger 거리 | > 500 |
| Cooldown | 6s |
| AttackType | Smash (패링 불가) |
| GroggyDamage | 0 |
| Damage | 50 |
| HitStop | 0.15s |
| ReactionType | Knockback |
| 돌진 속도 | 기본 MoveSpeed × 3 |

**텔 (0.4s)**
- 자세를 낮추고 두 머리가 앞을 향해 나란히 정렬
- 잔상 이펙트(Niagara Trail) 예고 발생

**히트 프레임**
- 돌진 시작 시 플레이어 위치/방향 스냅샷 저장
- 저장된 위치로 보간 이동 (`FInterpTo`, 캡슐 스윕으로 히트 판정)
- 돌진 중 히트 시 Knockback 적용

**리커버리 (0.8s)**
- 돌진 종료 후 오버슈트로 보스가 잠깐 지나침
- 오버슈트 구간이 딜 찬스

**대응 의도**
- 측면으로 회피 → 보스가 지나친 뒤 후방 반격
- 정면 회피 시 실패 (추적 보간으로 따라옴)

---

### P2-2) Syntax Venom `AOE / Zone Control`

> "바닥에 독 장판을 깔아 플레이어의 행동 반경을 압박"

| 항목 | 수치/내용 |
| --- | --- |
| Trigger 조건 | 플레이어가 보스 근거리(< 300)에서 3초 이상 유지 시 |
| Cooldown | 10s |
| AttackType | AOE (이동 회피 권장) |
| GroggyDamage | 0 |
| Damage | 장판 초당 8 (2~3초 지속) |
| 스태미나 디버프 | 장판 위에서 스태미나 회복 딜레이 +1.0s |
| 장판 반경 | 각 장판 150, 최대 3개 동시 |

**텔 (0.7s)**
- 바닥에 들여쓰기 라인(`indent`)처럼 보이는 독색 경고 패턴 표시
- 경고 패턴이 서서히 채워지며 발동 시점 안내

**히트 프레임**
- 경고 영역에 독 장판 생성 (Decal + Collision Box)
- 장판은 2~3초간 지속, 위에 있으면 지속 피해 + 스태미나 회복 방해

**리커버리**
- 보스 자체 후딜은 짧음 (0.3s). 장판이 압박을 지속하므로 보스는 바로 다음 행동 가능

**대응 의도**
- 무지성 근접 딜링을 막고 "공간 읽기"를 요구
- 장판 사이 안전지대로 이동하며 락온 유지

## 10. 선택 패턴 아이디어 (Nice / 확장)

### P2-3) Constrict `Grab`

- **컨셉:** 보스가 플레이어를 몸으로 감아 연속 피해를 입힘
- **탈출:** QTE (연타 또는 방향키 입력) 또는 회피 타이밍
- **대응:** 텔 구간에 회피 성공 시 완전 무효화. 실패 시 큰 피해

### P2-4) Double Strike `Normal`

- **컨셉:** 두 머리가 서로 다른 방향에서 동시에 공격
- **대응:** 한 방향 패링 후 다른 방향 회피를 요구하는 고난이도 패턴

### 체력 20% 이하 Climax 패턴

- **컨셉:** 두 머리가 합체해 거대 일직선 돌진
- **대응:** 좌우 완전 회피 (정중앙 사망)
- **연출:** 화면 연출 강화, 슬로우 모션 예고
